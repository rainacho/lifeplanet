라플 인터랙션 기본규칙

아토믹디자인 접근방식과 같이 복잡한 애니메이션도 조직적으로 구성되어 있다.
일관성있는 인터랙션은 사용자에게 향상된 서비스경험을 제공하고, 개발자에게 쉽고 체계적으로 작업 할 수 있는 환경을 제공한다.

2. Basic Easing

  • stop_circleStandard Easing(ease in - ease out)

    • Element가 모션이 시작할때 부터 끝까지 보이는 경우에 사용한다.
    예) Tile expanding, Table rows sorting

  • stop_circleEnterance Easing(ease out)

    • 화면에 새로운 element가 추가될 경우에 사용한다. Element가 빠르게 나타나서 천천히 멈춤
    예) modal, toaster message, dropdown opening, toggle

  • stop_circleExit Easing(ease in)

    • 화면에서 element가 사라질 경우에 사용한다. Element가 빠르게 사라짐
    예) closing modal and toaster message

  • stop_circleLinear Easing

    • non-moving elements에 사용한다.
    예) opacity or color transitions

4. Basic Animation Patterns

  • stop_circleSlidIn-Up (position + fade in)
    The Object is moving up while fading in, using enterance easing.
  • stop_circleSlidIn-Down (position + fade in)
    The Object is moving down while fading in, using enterance easing.
  • stop_circleSlidOut-Up (position + fade out)
    The Object is moving up while fading out, using exit easing.
  • stop_circleSlidOut-Down (position + fade out)
    The Object is moving down while fading out, using exit easing.
