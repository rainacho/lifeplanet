인터랙션 기본규칙

사용자에게 일관성 있는 서비스경험을 제공하고, 개발자에게 쉽고 체계적으로 작업 할 수 있는 환경을 제공하기 위하여 인터랙션규칙을 다음과 같이 정의한다.

2. Easing

  • stop_circleStandard Easing(ease in - ease out)

    • Element가 모션이 시작할때 부터 끝까지 보이는 경우에 사용한다.
    e.g. Tile expanding, Table rows sorting

    - jQuery 사용법 : easeInOutQuart
    - CSS 사용법 : cubic-bezier(0.76, 0, 0.24, 1)

  • stop_circleEnterance Easing(ease out)

    • 화면에 새로운 Element가 추가 되는 경우에 사용한다. Element가 빠르게 나타나서 천천히 멈춘다.
    e.g. buttom sheets, dialog, toaster message, dropdown opening, toggle

    - jQuery 사용법 : easeOutQuart
    - CSS 사용법 : cubic-bezier(0.25, 1, 0.5, 1)

  • stop_circleExit Easing(ease in)

    • 화면에서 Element가 사라질 경우에 사용한다. Element가 빠르게 사라진다.
    e.g. closing buttom sheets, closing dialog, closing side navigation

    - jQuery 사용법 : easeInQuart
    - CSS 사용법 : cubic-bezier(0.5, 0, 0.75, 0)

  • stop_circleLinear Easing

    • 움직이지 않는 Elements에 사용한다.
    e.g. opacity or color transitions

4. Animation Patterns

  • stop_circleSlideIn-Up (position + fade in)
    오브젝트가 밑에서 위로 나타나는 경우 Enterance Easing(ease out) 사용
  • stop_circleSlideIn-Down (position + fade in)
    오브젝트가 위에서 밑으로 나타나는 경우 Enterance Easing(ease out) 사용
  • stop_circleSlideOut-Up (position + fade out)
    오브젝트가 밑에서 위로 사라지는 경우 Exit Easing(ease in) 사용
  • stop_circleSlideOut-Down (position + fade out)
    오브젝트가 위에서 밑으로 사라지는 경우 Exit Easing(ease in) 사용
  • stop_circleScaleIn-Up (scale + fade in)
    오브젝트 크기가 커지면서 나타나는 경우 Entering Easing(ease out) 사용
  • stop_circleScaleIn-Down (scale + fade out)
    오브젝트 크기가 작아지면서 사라지는 경우 Exit Easing(ease in) 사용
